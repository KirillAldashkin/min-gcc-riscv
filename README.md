# Компиляция C/C++ в сырой код RISC-V
Работающий без дополнительной настройки компилятор GCC под RISC-V. За основу
взята [эта версия](https://github.com/xpack-dev-tools/riscv-none-elf-gcc-xpack),
которая была многократно облегчена и укомплектована готовыми скриптами.
## Готовые скрипты
### `compile`
Компилирует несколько файлов С++ в единственный файл `output.bin`,
содержащий только двоичный код. 
* Все аргументы передаются напрямую компилятору g++, можно указывать любые его флаги, 
например `-O3` для оптимизации по скорости или `-Os` для оптимизации по размеру.
* Стандартная библиотека отсутствует! Полностью!
* Не забудьте указать набор расширений, например флаг `-march=rv32im` установит 
базовую архитектуру RV32I с расширением M.
### `disasm`
Для заданного двоичного файла выводит его шестнадцатеричное и ассемблерное представление.
